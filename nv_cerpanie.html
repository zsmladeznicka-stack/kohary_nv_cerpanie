Jednorazové nastavenie

Pre prístup k súkromným tabuľkám Google potrebujete bezplatné Google OAuth Client ID.

1
Prejdite na Google Cloud Console → Credentials (vytvorte projekt ak treba — je to bezplatné).
2
Kliknite na + Create Credentials → OAuth client ID. Vyberte typ Web application.
3
V časti Authorised JavaScript origins pridajte URL tejto stránky (napr. http://localhost:8080).
4
Povoľte Google Sheets API aj Google Drive API v API Library.
5
V OAuth consent screen → Scopes pridajte: drive.readonly a spreadsheets (read/write).
6
Skopírujte Client ID (končí na .apps.googleusercontent.com) a vložte ho nižšie.
Váš Client ID je uložený iba v localStorage vášho prehliadača.
